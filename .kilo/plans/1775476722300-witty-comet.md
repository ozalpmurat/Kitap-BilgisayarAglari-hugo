# Plan: Görsel Yükleme Sorunu Çözümü

## Problem
Görseller yüklenmiyor çünkü:
- Görseller `content/BilgisayarAglari/images/` altında
- Markdown dosyaları `images/filename.png` yolunu kullanıyor
- Hugo'da bu yol doğru çalışmıyor (static klasöründe değil veya page bundle değil)

## Çözüm: Page Bundle Yapısı

### Adımlar:
1. Her bölüm .md dosyasını klasöre dönüştür (index.md olarak)
2. Görselleri ilgili bölüm klasörlerine taşı
3. Markdown dosyalarındaki görsel yollarını güncelle

### Örnek Dönüşüm:
```
Önce:
  content/BilgisayarAglari/02-Topolojiler.md
  content/BilgisayarAglari/images/B02-Topolojiler.jpg

Sonra:
  content/BilgisayarAglari/02-Topolojiler/index.md
  content/BilgisayarAglari/02-Topolojiler/images/B02-Topolojiler.jpg
```
